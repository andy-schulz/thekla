# Hover
Move the mouse pointer over an element.
 

## Ability

- [BrowseTheWeb](../../abilities/BROWSE_THE_WEB.md)

## Methods

| name       | parameter | description                     |
| :---       | :---      | :---                            |
| `.over()`* | -         | move the mouse over the element |

## Example

 ```typescript
 Josh.attemptsTo(
     Hover
         .over(Googles.IMAGINARY_HOVER_ELEMENT)
 )
```
